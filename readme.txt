Bibliotecas nescessárias para a execução do programa.

Abra o Prompt de Comando:
No Windows, pressione a tecla Win + R, digite "cmd" e pressione Enter.

Certifique-se de ter o Python instalado:
Antes de tudo, você precisa ter o Python instalado em seu sistema. Você pode verificar se o Python está instalado digitando python --version no prompt de comando.
Se você não tiver o Python instalado, precisará fazer o download e instalá-lo.

Instale as bibliotecas usando o pip:

pip install Pillow
pip install tabula-py
pip install PyPDF2
pip install pandas

Ainda no prompt de comando acesse a pasta do programa digite python robo_araya.py e aperte enter para iniciar o programa.
